## Stage_01/main/其他记号

### 大Ω记号
T(n) = Ω( f(n) ):
* 存在 c > 0 , 当 n >> 2 后，有T(n) > c * f(n)
###### 大Ω记号与大O记号正好相反，大Ω记号表示的是一种乐观预测(最优情况)

### 大Θ记号
T(n) = Θ( f(n) )
* 存在 c1 > c2 > 0，当n >> 2 后，有 c1 * f(n) > T(n) > c2 * f(n)
###### 大Θ记号是一种综合了大O和大Ω后的考量，表示的是一种平均的预测(平均情况)
